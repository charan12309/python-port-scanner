# Python Port Scanner

A simple and efficient TCP port scanner written in Python.  
This tool scans a specified range of ports on a target host and identifies open TCP ports.

> ⚠️ This project is intended for educational purposes and authorized testing only.

---

## Features
- TCP port scanning
- Multithreaded scanning for speed
- Configurable target and port range
- Clean and readable output
- Uses Python standard libraries only

---

## Requirements
- Python 3.7+
- No external dependencies
